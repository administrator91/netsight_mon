# Security Policy

## Authorized Use Only

NetSight Monitor is intended for educational, academic, lab, and authorized network monitoring environments only. Do not use it on networks, systems, or devices without clear permission.

## Reporting Issues

If you find a bug, crash, or security concern in this project, open a GitHub issue with:

- Operating system and Python version
- Network interface used
- Steps to reproduce
- Screenshot or error message if available

Do not include real credentials, private IP details, personal information, or sensitive packet data in public issues.
