# Ethical Use Guidelines

NetSight Monitor is a learning and monitoring tool. Use it only in environments where you have permission.

## Allowed Use

- Final-year project demonstration
- Personal lab testing
- Authorized local network monitoring
- Security awareness training
- Protocol learning and troubleshooting

## Not Allowed

- Capturing traffic from networks without permission
- Collecting or publishing real credentials
- Sharing private packet captures publicly
- Using the tool for unauthorized surveillance

## Recommended Practice

Use demo websites, test credentials, and lab networks. Blur or remove sensitive values before adding screenshots to reports, GitHub, or presentations.
